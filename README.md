# 🌟 SILLORA Perfume Store

موقع احترافي لبيع عطر SILLORA مع لوحة تحكم أدمن كاملة.

## 📁 ملفات المشروع

```
sillora-perfume/
├── server.js          # الخادم (Node.js + Express + SQLite)
├── package.json       # تبعيات المشروع
├── database.sqlite    # قاعدة البيانات (تُنشأ تلقائياً)
└── public/
    ├── index.html     # صفحة المتجر (الواجهة الأمامية)
    └── admin.html     # لوحة تحكم الأدمن
```

## 🚀 طريقة التشغيل

### 1. تثبيت Node.js
تأكد من تثبيت Node.js من: https://nodejs.org

### 2. تثبيت التبعيات
افتح Terminal/Command Prompt داخل مجلد المشروع واكتب:
```bash
npm install
```

### 3. تشغيل السيرفر
```bash
npm start
```

### 4. فتح الموقع
- المتجر: http://localhost:3000
- لوحة الأدمن: http://localhost:3000/admin.html

## 🔐 بيانات دخول الأدمن
